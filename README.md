### Installation
```bash
# Clone the repository
git clone https://github.com/innaesim/portfolio.git

# Navigate to project folder
cd portfolio

npm install
````

### Development

```bash
# Start the local development server
npm run dev
```

### Build for Production

```bash
npm run build
```
